# 知乎日报 简易版
这是一个简易版的知乎日报。
实现了以下功能：
* 下拉刷新(SwipeRefreshLayout)
* 内容阅读(WebView以及相应CSS的应用)
* listview使用卡片式UI(layer-list)
* 收藏功能


## Dependency
* gson-2.3.1
* universal-image-loader-1.9.3
